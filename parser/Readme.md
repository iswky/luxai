# 🚀 Система парсинга тендеров с портала zakupki.gov.ru

## 📋 Общее описание

Система состоит из двух независимых скриптов для автоматического сбора информации о государственных закупках:

1. **`parse_gos_zalupki.py`** — парсинг списка тендеров и сохранение в Excel
2. **`Zalupki_files_download.py`** — скачивание файлов и определение города заказчика

---

## 📁 Структура данных

### Excel файл: `tenders.xlsx`

| Колонка | Название | Описание |
|:-------:|----------|----------|
| A | Номер тендера | Уникальный идентификатор закупки |
| B | Название/Объект закупки | Краткое описание предмета закупки |
| C | Ссылка на тендер | URL на страницу с общей информацией |
| D | Цена (руб) | Начальная (максимальная) цена контракта |
| E | ФЗ | Закон регулирования (44-ФЗ или 223-ФЗ) |
| F | Окончание подачи заявок | Дата и время окончания приема заявок |
| G | Заказчик | Наименование организации-заказчика |
| H | Дата добавления | Время добавления записи в базу |
| I | Город | Местоположение заказчика |
| J | Файлы скачаны | Статус скачивания документов (`True`/`False`) |
